인광기 서비스에 다중 브랜드 언어 비교 뷰(brand-compare.html)를 추가하기로 결정. 기존 단일 브랜드 뷰(numbuzin-final.html)와 독립된 별도 진입점으로 운영.

**Why:** 대기업 마케팅 팀(주 사용자)이 경쟁 브랜드 언어 지형을 카테고리 단위로 조망하고, 아무도 쓰지 않는 언어 공백(white space)을 발견해 캠페인 방향에 활용하도록.

**How to apply:** 아래 결정사항을 구현 기준으로 삼는다.

## 확정된 설계 결정사항

| 항목 | 결정 |
|------|------|
| 주 사용자 | 대기업 마케팅 팀 (기존 B2B 파트너) |
| 핵심 기능 | 언어 공백(white space) 발견 |
| 핵심 시각화 | **점령 바** — 패턴별 브랜드 점유율, 빈 공간=공백 |
| 블럭 구성 | 점령 바(full width) → 하단: 중복단어랭킹(좌) + 크리에이터추천/AI언어방향(우) |
| 브랜드 선택 | 카테고리 프리셋 탭 + 수동 칩 추가 |
| 레이아웃 | 점령 바 full width → 아래 좌우 분할 (360px 고정 | flex) |
| 단일 뷰 연결 | 점령 바 세그먼트 클릭 → 우측 슬라이드오버 패널 |
| 패널 내부 | 기존 numbuzin 대시보드 콘텐츠, 포스트 그리드 3열, 간격 축소 |
| 패널 너비 | clamp(600px, 62vw, 900px) — 구현 시 조정 가능 |
| 플랫폼 연결 | 크리에이터 추천에 버튼만, 링크는 나중에 |
| AI 출력 레벨 | 레벨 2 — 언어 포지셔닝 전략 (카피 아님) |
| 시계열 | 보류 (데이터 없음, v2 과제) |
| 저장 기능 | 목업이므로 불필요 (새로고침 시 초기화) |

## 데이터 구조
- 프리셋: 뷰티 세럼(기본), 선케어, 클렌징
- 비교 브랜드: 넘버즈인(#E84A6F), 마녀공장(#7C5CBF), 아누아(#4A9E6F), 토리든(#3A86C8), 클리오(#C44D6E)
- 패턴: 감각형의태어(92%), 사회적증거(77%), 긴급성프로모션(80%), 트렌드바이럴(48%→공백), 내부자정보(25%→대공백)
- 중복단어: 포화(세럼/모공/피부결/촉촉/올리브영) / 공백(임상결과/피부과추천/전문가인증)
- AI 언어 방향: 내부자정보 75% 공백 + 트렌드바이럴 52% 공백 선점 전략 제안
