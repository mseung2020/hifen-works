# 브랜드 언어 비교기 (Language Analysis)

경쟁 브랜드들의 인스타 광고 캡션을 **카테고리별로 나란히 놓고, 어떤 표현 패턴(감각·기능·일상·사회적 증거·트렌드)에서 누가 강세이고 어디가 비어있는지(언어 공백, white space)**를 한눈에 보여주는 마케팅 도구 프로토타입입니다. 대기업 마케팅 팀이 경쟁 브랜드의 언어 지형을 조망하고 아무도 쓰지 않는 표현 공백을 찾아 캠페인 방향을 잡도록 돕는 것이 목표입니다.

- **분야:** 데이터 분석
- **결과물 형태:** 단일 HTML 프로토타입 (다중 브랜드 비교 뷰)
- **핵심 시각화:** **점령 바** — 표현 패턴별 브랜드 점유율, 빈 공간 = 언어 공백

## 결과물

[`brand-compare.html`](brand-compare.html) — **최종 다중 브랜드 언어 비교 뷰**. 브라우저로 바로 열람. 구성:
- **점령 바(full width):** 패턴별 브랜드 점유율 시각화 (빈 공간 = 공백)
- **하단 좌:** 중복 단어 랭킹 (포화 단어 vs 공백 단어)
- **하단 우:** 크리에이터 추천 + AI 언어 방향 제안 (언어 포지셔닝 전략 레벨)
- 점령 바 세그먼트 클릭 → 단일 브랜드 상세 슬라이드오버 패널 (포스트 그리드)

> 브랜드 로고·크리에이터 사진은 원격 URL을 사용하고, 포스트 썸네일은 포함된 `images/posts/`(203장)를 참조합니다. 카테고리 프리셋(뷰티 세럼/선케어/클렌징)과 비교 데이터는 HTML에 내장된 목업 데이터입니다.

## 구성

| 파일 | 설명 |
|---|---|
| `brand-compare.html` | 최종 다중 브랜드 언어 비교 프로토타입 |
| `images/posts/` | 비교 뷰에 표시되는 포스트 썸네일 (203장) |
| `기획_설계.md` | 비교 뷰의 기획 결정사항·설계 근거·데이터 구조 정의 |
| `dify_language_analysis.yml` | 캡션 언어 패턴 분석용 Dify 워크플로우 |
| `word_frequency.py`, `prepare_captions_input.py` | 캡션 단어 빈도 집계·입력 전처리 스크립트 |

> 이 폴더는 단일 브랜드 뷰·여러 목업을 거쳐 다중 브랜드 비교 뷰(`brand-compare.html`)로 수렴한 작업으로, 여기에는 최종본만 담았습니다.
