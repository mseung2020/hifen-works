# 인스타 크리에이터 커스텀 랭킹 프롬프트

검색 필터 조합을 분석해 크리에이터 **랭킹 리스트 타이틀 + 3문단 설득 설명글**을 실시간으로 자동 생성하는 AI 프롬프트입니다. 광고주/마케터가 필터 검색을 하면, 결과 상단에 카드형으로 노출되는 큐레이션 카피를 LLM이 즉석에서 만들어 줍니다.

- **분야:** AI / 프롬프트
- **플랫폼:** DIFY (AI 워크플로우)
- **모델:** GPT-4 mini (가벼운 실시간 생성용)
- **출력:** `{ "ranking_title": ..., "ranking_description": ... }` JSON

## 입력 변수

| 변수 | 의미 | 역할 |
|---|---|---|
| `follower` | 팔로워 구간 (직접 입력 가능) | 체급 묘사 |
| `user_type` | 유저 타입 | 톤 보조 |
| `content` | 콘텐츠 성격 | 슬롯C 페르소나 결정 |
| `field` | 분야(20종) | 슬롯B 타겟 키워드 결정 |
| `mood` | 무드(11종) | 슬롯A 감성 수식구 결정 |
| `appearance` | 출연 형태 | 보조 |
| `tag` | 해시태그 | 슬롯B 우선 반영 (없으면 무시) |
| `sort` | 정렬 기준 | 강점 묘사 |

## 동작 원리

1. **타이틀 의사결정 트리** — 타이틀(공백 포함 최대 30자)을 3개 슬롯으로 구성
   - `[A: 감성 수식구]` + `[B: 타겟 키워드]` + `[C: 페르소나]`
   - C(페르소나)는 반드시 맨 끝에 위치
2. **변수 맵핑 사전** — 팔로워 체급 변환표(미들/매크로/메가/슈퍼메가/글로벌 아이콘)와 정렬 기준 강점 변환표를 설명란에 적용
3. **제한 사항** — 30자 제한, 이모지 금지, 할루시네이션 금지, 팔로워 숫자 노출 금지, 리스트 외 표현 금지 등
4. **"전체 선택" 폴백 로직** — 변수가 "전체"일 때 다른 변수에서 톤을 유추
5. **Few-shot 4종** — 기본 케이스 / 태그 복수 / 전체 선택 / 직접 입력 범위
