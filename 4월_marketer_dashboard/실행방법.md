# 대시보드 실행 방법

## 1. 로컬 서버만 켜기 (본인만 확인)

```bash
python3 app.py
```

→ http://localhost:5000 에서 확인

## 2. 로컬 서버 + 외부 공유 링크 동시에 켜기

```bash
python3 app.py &
cloudflared tunnel --url http://localhost:5000
```

→ 터미널에 `https://xxxxx.trycloudflare.com` 주소가 뜨면 그걸 공유

## 3. 전부 끄기

```bash
pkill -f "python3 app.py"
pkill -f cloudflared
```

## 4. PythonAnywhere (24시간 무료 호스팅)

**주소:** https://mseung2020.pythonanywhere.com

### 코드 수정 후 반영하는 법

1. 로컬에서 수정 완료
2. 다시 zip 만들기:
```bash
cd /Users/rachel/Documents/김명승\ 작업실/4월_marketer_dashboard
zip -r mysite.zip app.py config.json requirements.txt templates/ static/ data/
```
3. PythonAnywhere → **Files** 탭 → `mysite.zip` 재업로드
4. **Consoles** → Bash 열고:
```bash
cd ~ && unzip -o mysite.zip -d mysite
```
5. **Web** 탭 → **Reload** 클릭
6. 끝!

### 주의사항
- **절대 Web 탭에서 "Add a new web app"을 다시 누르지 말 것** (app.py가 덮어씌워짐)
- 파일만 교체 + Reload로 충분함
