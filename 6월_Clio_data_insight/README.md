# Clio 브랜드 데이터 인사이트 리포트

클리오·페리페라 브랜드의 **발행량·올리브영 랭킹·조회수**를 CSV로 정리하고 HTML 시각화로 마감한 간이 리포트입니다. 뷰티 유튜브·인스타 조회수 TOP100 브랜드를 기준으로 클리오·페리페라의 위치와 성과 흐름을 보여줍니다.

- **분야:** 데이터 분석
- **결과물 형태:** 단일 HTML 시각화 세트 (자체 완결형)

## 결과물 (HTML)

| 파일 | 내용 |
|---|---|
| `브랜드_4분면.html` | 브랜드 4분면 포지셔닝 |
| `조회수_vs_올리브영랭킹.html` | 조회수 vs 올리브영 랭킹 관계 |
| `클리오_월별발행량.html` | 클리오 월별 발행량 |
| `클리오_페리페라_월별발행량.html` | 클리오·페리페라 월별 발행량 비교 |
| `클리오_페리페라_월별발행량_로고.html` | 위와 동일 (브랜드 로고 표기 버전) |

> 모든 HTML은 데이터가 내장된 자체 완결형이라 브라우저로 바로 열람됩니다.

## 데이터 & 생성 스크립트

| 파일 | 설명 |
|---|---|
| `_gen_viz.py` | CSV → 시각화 데이터(`_viz_data.json`) 생성 스크립트 |
| `_viz_data.json` | 시각화용 가공 데이터 |
| `26상반기_뷰티 유튜브/인스타 조회수 top100 브랜드.csv` | 뷰티 조회수 TOP100 브랜드 |
| `전체 브랜드 랭킹.csv` | 전체 브랜드 랭킹 |
| `클리오 월별 발행량.csv`, `페리페라 월별 발행량.csv` | 월별 발행량 |
| `클리오 제품.csv`, `클리오_올리브영_일간랭킹.csv` | 클리오 제품·올리브영 일간 랭킹 |
