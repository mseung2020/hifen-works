# LG 광고시장 데이터 인사이트 리포트

광고·올리브영·크리에이터 데이터를 끌어와 **회귀·상관·시계열로 분석**하고, 그 결과를 만화 리포트·카드뉴스·특별취재 같은 **읽기 좋은 HTML 리포트 패키지**로 마감한 대규모 데이터 분석 프로젝트입니다. 핵심 질문은 *"올리브영 랭킹을 만드는 진짜 동인은 무엇인가"*입니다.

- **분야:** 데이터 분석
- **결과물 형태:** 만화 리포트·카드뉴스·특별취재 HTML 패키지 + 분석 리포트(MD) + 최종 결론
- **핵심 발견:** 랭킹 동인은 '물량·광고비'가 아니라 **'체급별 효율·참여'** (→ [최종결론.md](최종결론.md))

## 🏁 최종 결론
[`최종결론.md`](최종결론.md) — 크리에이터 체급(나노~메가)별로 올리브영 랭킹과 연결되는 지표를 4개 사례(Spearman 순위상관)로 정리. **메가 물량 베팅은 랭킹 동인이 아니었고**, 유튜브는 미드티어의 구독자 대비 조회 효율, 인스타는 미드티어 광고 도달 + 나노의 참여율이 핵심.

|---|---|---|

| 만화 | `만화리포트_EP3_발행량통제.html` | 발행량(콘텐츠 물량) 통제 분석 EP3 |
| 카드뉴스 | `카드뉴스_올리브영_분포.html` | 올리브영 랭킹 분포 |
| 카드뉴스 | `카드뉴스_올리브영_트리맵.html` / `_공유용.html` | 올리브영 트리맵 시각화 |
| 특별취재 | `올리브영_유튜브_특별취재.html` | 올리브영×유튜브 심층 취재형 리포트 |
