# 데이터 요약

이 프로젝트는 광고·올리브영·크리에이터 데이터를 다각도로 분석한 대규모 작업으로, **원천/중간 데이터 총량이 약 6GB**입니다. 최종 결과물(리포트·결론)만 레포에 담고, 원천 데이터는 아래로 요약합니다.

## 주요 원천 데이터 (제외)
| 데이터 | 규모 | 내용 |
|---|---|---|
| `creators.csv` | 469MB | 인스타 크리에이터 전체 |
| `youtube_creators.csv` | 291MB | 유튜브 크리에이터 전체 |
| `카테고리_raw/` | 192MB | 카테고리별 raw 데이터 |
| `마스터데이터/` | 182MB | 분석용 마스터 데이터 |
| `최종 데이터/` | 162MB | 정제 완료 분석 데이터 |
| `올영 랭킹/`, `최종 올영 랭킹/` | 180MB | 올리브영 랭킹 시계열 |
| `유저 폴더/` | ~4GB | 유저 단위 raw |
| 그 외 | — | 시계열분석, 브랜드크리_분석, 인지도 분석, 제품_피처_데이터 등 중간 산출물 |

## 분석 방법
- DB 스키마 정리 + 로컬 적재로 분석 환경 구축
- 브랜드·크리에이터·올리브영 랭킹·인지도·시계열 등 **회귀·상관·시계열** 다각도 분석
- 핵심 지표: 크리에이터 체급(나노~메가)별 효율·참여율 vs 올리브영 TOP10/TOP100 진입 (Spearman 순위상관)

> 분석 코드와 중간 산출물(analysis/, DB_스키마_AI참조.md 등)은 원본 작업실에 보존되어 있으며, 이 레포에는 읽기 좋은 최종 리포트와 결론만 담았습니다.
