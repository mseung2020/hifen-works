# 데이터 요약

원본 크롤링 데이터는 244MB~855MB로 GitHub(파일당 100MB)에 올릴 수 없어, **스키마·규모·통계·샘플**로 요약해 둡니다.

## 포함된 파일
- `ad_insta_id.csv` (2MB, 전체 포함) — 광고로 라벨된 게시물 목록. 컬럼: `id;post_id` · 총 96,992행. recall(광고 보존율) 계산 기준
- `total_insta_id_slim.sample.csv` — 전체 크롤링 데이터의 **앞 99행 샘플** (형식 확인용)

## 제외된 원본: `total_insta_id_slim.csv`
- **규모:** 16,506,692행 (약 855MB), 구분자 `;`
- **컬럼:** `post_id, is_reel, views, likes, comments, followers, publish_date, is_sponsored, is_ppl`

### 분포 통계 (전체 16.5M행 기준)
| 항목 | 값 |
|---|---|
| is_reel = 1 (릴스) | 6,705,524건 (40.6%) |
| is_sponsored = 1 | 66,997건 (0.4%) |
| is_ppl = 1 | 483,879건 (2.9%) |

| 지표 | 평균 | 최댓값 |
|---|---|---|
| views | 39,428 | 501,199,067 |
| likes | 2,552 | 26,494,517 |
| comments | 40.4 | 1,519,884 |
| followers | 54,505 | 291,980,128 |

> 광고(sponsored/ppl) 게시물은 전체의 극소수(합쳐도 ~3%)이고 지표 분포가 매우 넓은(롱테일) 데이터입니다. 필터링 분석은 이 소수의 광고를 거의 놓치지 않으면서(recall ≥ 0.95) 비광고 다수를 걸러내는 임계값을 찾는 것이 목표였습니다.
