# 데이터 요약

Flask 앱은 `static/abib_data.json`(사전 가공 데이터)을 읽어 동작하므로, 아래 원천 CSV 없이도 실행됩니다.

## 포함된 데이터 (`data/`)
| 파일 | 설명 |
|---|---|
| `아비브 유튜브.csv` | 아비브 유튜브 영상 데이터 |
| `아비브 인스타.csv` | 아비브 인스타 데이터 |
| `아비브 제품.csv` | 아비브 제품 목록 |
| `아비브_올리브영_일간랭킹.csv` | 아비브 올리브영 일간 랭킹 |

## 제외된 대용량 CSV
`prepare_data.py`가 참조하는 범용 원천 데이터. 용량 절감을 위해 제외.

| 파일 | 크기 | 내용 |
|---|---|---|
| `올리브영_일간_랭킹_순위.csv` (+ _2, _3) | 각 21~25MB | 올리브영 전체 일간 랭킹 순위 |
| `제품 1년치 목록.csv` | 44MB | 1년치 제품 목록 |

> 흐름: 원천 CSV → `prepare_data.py`로 티어 분석 → `static/abib_data.json` 생성 → Flask 앱이 조회 리포트로 표시. 가공 결과(`abib_data.json`)는 포함되어 있어 앱이 바로 실행됩니다.
