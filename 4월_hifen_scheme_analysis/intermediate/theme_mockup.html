#빠른 시작

Atlas는 단일 HTML 파일로 배포되며 더블클릭만으로 열립니다. 외부 의존은 D3, Pretendard, JetBrains Mono 세 가지 CDN뿐입니다.

Tip
왼쪽 사이드바로 7개 뷰를 이동하고, 테이블명 또는 앱명을 검색창에 입력해 바로 드릴다운할 수 있습니다.

#핵심 개념

Atlas는 다음 세 가지 소스를 하나로 결합합니다. 각 뷰는 세 소스 중 관련된 슬라이스를 직접 참조합니다.

#재빌드

스냅샷을 최신으로 갱신하려면 intermediate 디렉토리의 스크립트를 순서대로 실행합니다.

bash # 1. 재스캔 & 재구성 python3 intermediate/label_tables.py python3 intermediate/extract_relations.py python3 intermediate/detect_unused_endpoints.py # 2. JSON 번들 python3 intermediate/build_site_data.py # 3. HTML 다시 빌드 python3 intermediate/build_html.py

빌드가 끝나면 core_materials/index.html을 열어 확인합니다.
