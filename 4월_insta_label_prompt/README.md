# 인스타 계정 신규 라벨링 프롬프트

인스타그램 계정을 마케팅 관점에서 분류하는 AI 프롬프트입니다. 프로필과 최근 게시물을 보고 계정 성격을 **6개 Layer로 나눠 0~100점**으로 점수화하고, 오디언스(성별·연령·국가)까지 추정합니다. 브랜드 마케터가 인플루언서를 선정할 때 참고할 분류 신호를 만드는 것이 목표입니다.

- **분야:** AI / 프롬프트
- **플랫폼:** DIFY (AI 워크플로우)
- **출력:** Layer별 점수·근거·primary_label을 담은 JSON

## 6개 Layer 체계

| Layer | 분류 | 예시 라벨 |
|---|---|---|
| 1 | 채널 타입 | 휴먼 인플루언서, 공식/브랜드, 미디어/파워페이지, 커머스 셀러, 커뮤니티 |
| 2 | 콘텐츠 기능 | 리뷰, 꿀팁/하우투, 일상/공감, 엔터테인먼트, 루틴, 프로모션, 큐레이션 |
| 3 | 분야/주제 | 뷰티, 패션, 푸드, 여행, 육아, 스포츠/피트니스, 반려동물 등 |
| 4 | 무드/톤 | 밝은, 자연스러운, 다크/무디, 럭셔리, 미니멀, 시크, 감성적 등 |
| 5 | 출연 구조 | 얼굴 중심, 목소리/자막 중심, 제품 중심, 가족/반려동물 중심, 편집형 |
| 6 | 예상 오디언스 | 타겟 국가(ISO 코드)·성별 비율·연령대 분포 추정 |

## 동작 원리

1. **신호 추출** — 이미지 신호를 우선 확인하고 텍스트로 보완 (복수 라벨 허용, 점수 합 ≠ 100)
2. **Layer별 점수화** — 근거(top_evidence) 기반으로 0~100점 부여, primary_label 선정
3. **오디언스 추정** — 언어·위치·상업·문화·메타 신호로 타겟 국가 점수화, 시각적 어필 신호로 성별/연령 비율 보정 (여행 콘텐츠는 타겟 국가에서 배제)
4. **confidence 산정** — Layer별 판단 확신도 부여
5. **JSON 출력** — 입력에 없는 사실 추측 금지, reason은 100자 이내
