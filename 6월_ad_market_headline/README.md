# 오늘자 광고시장 헤드라인

매일 광고시장에서 일어난 사건을 **1줄 헤드라인 10선 카드 피드**로 정리하는 마케터용 데일리 대시보드입니다. 유튜브 PPL·인스트림 광고·인스타그램 PPL 3개 채널 탭으로 전환하며, 채널별 브랜드 뷰·집행 횟수 TOP 위젯과 자세히 보기 슬라이드 모달을 제공합니다.

- **분야:** 데이터 분석
- **결과물 형태:** 단일 HTML 목업 대시보드
- **데이터 흐름:** 광고시장 raw 데이터 → LLM 입력 생성 → Dify 워크플로우가 헤드라인 10선 생성 → 대시보드 표시

## 결과물

[`목업_헤드라인.html`](목업_헤드라인.html) — **최종 헤드라인 데일리 대시보드 목업**. 브라우저로 바로 열람. 구성:
- 오늘의 헤드라인 10선 카드 피드 + 자세히 보기 슬라이드 모달
- 유튜브 PPL / 인스트림 광고 / 인스타그램 PPL **3개 채널 탭** 전환
- 채널별 브랜드 뷰·집행 횟수 TOP 위젯

> 포스트 썸네일은 포함된 `images/posts/`를 참조하고, 헤드라인·집계 데이터는 HTML에 내장돼 있습니다.

## 파이프라인 (`pipeline/`)
헤드라인을 만들어내는 LLM 파이프라인 자산:

| 파일 | 설명 |
|---|---|
| `download_today_queries.sql` | 당일 광고시장 데이터 추출 SQL |
| `build_llm_input.py` | raw 데이터를 LLM 입력 JSON으로 가공 |
| `dify_system_prompt_v5.md` | 헤드라인 생성 Dify 시스템 프롬프트 (최종 v5) |
| `llm워크플로우_v2.yml` | Dify 워크플로우 export (v2) |
| `headlines_2026-06-01.md` | 생성된 헤드라인 결과 예시 |
| `파이프라인_검증결과.md` | 파이프라인 검증 기록 |

## 기타
- `기획서.md` — 광고시장 헤드라인 대시보드 기획서
- `data/` — 파이프라인 입력 JSON + 제외된 대용량 raw 데이터 요약 ([DATA_SUMMARY.md](data/DATA_SUMMARY.md))

> 매거진/카드/랭킹 등 여러 디자인 시안을 거쳐 헤드라인 카드 피드(`목업_헤드라인.html`)로 확정했으며, 여기에는 최종본만 담았습니다.
