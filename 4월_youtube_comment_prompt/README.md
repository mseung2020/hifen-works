# 유튜브 댓글 기반 시청자 분석 검증 프롬프트

유튜브 채널의 **시청자 성별·연령대 분포를 댓글 데이터로 검증·개선**하는 AI 프롬프트입니다. 기존 LLM이 도출한 시청자 분석값(demography)이 실제와 맞는지 댓글 신호로 교차검증하고, 이상이 감지되면 분포를 재분배해 개선값을 제공합니다.

- **분야:** AI / 프롬프트
- **플랫폼:** DIFY (AI 워크플로우)
- **입력:** 기존 분석값(demography) + 채널/콘텐츠 설명 + 댓글 데이터(JSON)
- **출력:** `anomaly_detected`, `anomaly_reason`, `improved_values`(14개 성별×연령 구간) JSON

## 동작 원리

1. **댓글 신호 식별** — 작성자명·댓글 텍스트에서 성별/연령 신호를 추출해 카운트 (예: "형님/아재"→남성, "언니/오빠"→여성)
2. **역신호 함정 처리** — 크리에이터 성별 ≠ 시청자 성별. 여성 아이돌 외모 평가 댓글은 *남성 시청자* 신호로 해석
3. **이상 징후 감지** — ① 콘텐츠 유형이 시사하는 연령대 vs 기존값 비교 ② 댓글 신호 vs 기존값 비교. 10%p 이상 차이 시 anomaly
4. **콘텐츠 유형별 분포 프라이어** — 게임=우하향 감소형, K-pop=양봉형, 트로트/낚시=우상향 등 정상 분포 형태 적용 (정규분포 강제 금지)
5. **재분배 + 필수 검산** — 총합 100 유지, anomaly_reason과 improved_values 방향 일치 확인 등 5단계 검산 통과 후에만 출력

## 파일

| 파일 | 설명 |
|---|---|
| `prompt.md` | 최종 프롬프트 (시청자 분석 검증 + 이상감지 + 재분배) |
| `dify_workflow.yml` | DIFY 워크플로우 export |
| `sample_comments/` | 입력 댓글 데이터 샘플 (영상별 댓글 JSONL, 일부만 발췌) |

> 본 작업은 댓글 라벨링·오디언스 분석·이상감지 등 여러 서브프롬프트를 거쳐 위 검증 프롬프트로 통합되었습니다. 원본 샘플 데이터는 16,000여 개 영상 분량으로, 여기에는 형식 예시용 일부만 포함했습니다.
