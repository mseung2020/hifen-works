# 쿠폰 시스템 코드 분석

하이픈 백엔드(`ugwanggiAPI`)에 **미개발 상태로 남아있던 쿠폰 코드를 분석**하고, 쿠폰 시스템을 어떻게 개편·개발하면 되는지 연구한 작업입니다. 백엔드 레포를 직접 들여다보며 현재 쿠폰 테이블 구조와 결제 흐름을 파악하고, 퍼센티지 할인 도입을 위한 구체적인 코드 수정 가이드로 정리했습니다.

- **분야:** 백엔드 / 코드
- **분석 대상:** `ugwanggiAPI` (feature-new_coupon 브랜치) — Django 백엔드
- **핵심 발견:** 퍼센티지 할인용 `CustomCouponSettings2` 테이블이 모델만 정의되고 결제 로직과 미연결된 상태 → "잠든 테이블을 깨우는" 개편안 제시
- **도구:** MCP 연결로 노션에 분석 결과 정리
