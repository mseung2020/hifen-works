INSIGHT 02 · 유가 크리에이터 체급

‘상위 제품 = 나노 위주’는 통념일 뿐, 뚜렷하지 않다

각 제품을 광고(유가)한 크리에이터(유튜브 채널+인스타 계정, 중복 제거)의 체급 구성 — 전체 보드 Top100 유지일수 순 정렬. x축 아래 = 크리에이터 수 · 유지일수.

체급 = 유튜브 구독자·인스타 팔로워 기준 7단계(나노 <1만 / 마이크로 1~5만 / 미드티어 5~10만 / 하이티어 10~20만 / 매크로 20~50만 / 메가 50~100만 / 셀럽 100만+). 유가 = YT ppl · IG sponsored/ppl. 미분류 = 크리에이터 메타 없음(대부분 초소형). 기간 2025-09~2026-06(인스타 2026-03~).
