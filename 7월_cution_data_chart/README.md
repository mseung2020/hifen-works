# 쿠션 제품 시장 데이터 리포트

메이크업 쿠션 시장을 **발행량·올리브영 랭킹·인플루언서 분산도·릴스 비중·광고 성과** 등 여러 각도로 분석하고, **클리오가 정샘물과 어깨를 나란히 하고 있다**는 점을 데이터로 짚어낸 3장짜리 내러티브 리포트입니다. 분석 도표는 직접 만든 **커스텀 차트 웹툴**로 생성했습니다.

- **분야:** 데이터 분석
- **결과물 형태:** 내러티브 리포트(3장) + 인사이트 차트 HTML 세트 + 커스텀 차트 웹툴

## 🏁 리포트 본문
[`리포트.md`](리포트.md) — 3장짜리 내러티브 (시장 큰 흐름 → 브랜드별 진단 → 클리오 제언). 핵심 발견:
- 발행량 vs 올리브영 순위 상관 **−0.73** (많이 낼수록 순위 높음)
- 인플루언서는 "1인 1게시물"로 넓게 분산 / 인스타는 릴스 80~100%(정샘물만 예외)
- **클리오는 물량·광고 성과(평균 조회수 대비 51.7%, 9개 브랜드 중 1위) 모두 이미 최상위권**

## 📊 인사이트 차트 (`charts/`)
분석 결과 도표 27종 (자체 완결형 HTML, 브라우저로 바로 열람). 발행량-순위 산점도, VDL/정샘물/파넬 주별 흐름, 클리오 유가·물량, 4분면, 전환율, 티어 비교 등.

## 🛠 커스텀 차트 웹툴 (`chart_webtool/`)
도표 생성에 사용한 웹 툴 소스. CSV/엑셀을 올리면 막대·선 그래프를 그리고 **투명 배경 PNG로 내보내는** React + Vite + TypeScript + ECharts 앱.
```
cd chart_webtool && npm install && npm run dev   # http://localhost:5173
```
> `node_modules`·빌드 산출물은 제외했습니다. 분석 원천 데이터(브랜드별 발행량·올리브영 랭킹 CSV 등)와 40여 개 `build_*.py` 생성 스크립트는 용량 관계로 원본 작업실에 보존하고, 여기에는 최종 결과물(리포트·차트·웹툴)만 담았습니다.
